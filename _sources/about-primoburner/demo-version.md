# Demo Version

## Restrictions

When operating in demo mode, read / write speeds are limited to:

- 8x for CD-R, and CD-ROM
- 4x for CD-RW
- 4x for DVD-R, DVD+R, and DVD-ROM
- 2x for DVD+RW and DVD-RW
- 1x for Blu-ray Disc
